# EditorVideoIA - Correção Salvar + F5

Correção para persistência da timeline ao apertar F5.

Inclui:
- Controller completo com normalização da timeline.
- Model completo do projeto.
- View completa com backup local e salvamento no banco.
- Migration segura para garantir colunas necessárias.
- Rotas completas do EditorVideoIA.

Instalação:
1. Extraia tudo na raiz do projeto: C:\Projetos\EditorVideoIA\backend
2. Rode: INSTALAR_CORRECAO_SALVAR_F5.bat
3. Rode: php -S 127.0.0.1:8020 -t public public/index.php
4. Abra: http://127.0.0.1:8020/editor-video
